# How to use
````
$ mkdir <repos name>
$ cd <repos name>
$ python3 -m venv venv
$ source venv/bin/activate
````

### Clone this project

````
$ git clone git@github.com:Nsabirzhanov/MyFirstRep.git
$ pip install -r requirements.txt
````

### Finally run this project
````
$ python3 manage.py runserver
````
